# Oneming Roadmap

## Current Phase

Oneming is moving from a single AI fortune-telling tool to an AI-powered Chinese Metaphysics knowledge platform.

Current focus:

- build the product foundation
- standardize the knowledge system
- create reusable knowledge objects
- connect tools, reports, guides, and knowledge pages

## Sprint 1: Foundation

Goal:

Build the project foundation before adding new features.

Tasks:

- Create Product Bible
- Create Knowledge Spec
- Create Roadmap
- Create SEO principles
- Create Architecture document
- Create first 50 knowledge objects
- Define URL structure
- Define internal linking rules

## Sprint 2: Knowledge System

Goal:

Turn core metaphysics concepts into structured knowledge pages.

Tasks:

- Complete 50 core knowledge objects
- Build /knowledge homepage
- Build /knowledge/:slug page template
- Add related concepts
- Add FAQ sections
- Add SEO metadata
- Add internal links from report pages to knowledge pages

## Sprint 3: Guides

Goal:

Create beginner-friendly learning paths.

Tasks:

- Create /guides homepage
- Publish first 10 guides
- Link guides with knowledge objects
- Add “Learn more” paths from AI reports

## Sprint 4: AI Knowledge Engine

Goal:

Make AI reports reference Oneming knowledge.

Tasks:

- Connect report generation with knowledge objects
- Add cited explanations
- Add recommended reading
- Improve trust and clarity

## Sprint 5: Growth

Goal:

Expand search visibility without low-quality mass pages.

Tasks:

- Create zodiac pages
- Create birthday pages gradually
- Improve sitemap
- Improve Search Console coverage
- Improve user page depth
