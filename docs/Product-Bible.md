# Oneming Product Bible v1.0

## 1. Vision

Oneming is an AI-powered Chinese Metaphysics platform.

It is not only a fortune-telling tool.  
It is a structured knowledge system that helps users understand Chinese metaphysics through AI, knowledge, and guided learning.

## 2. Positioning

Oneming = Chinese Metaphysics Knowledge Graph + AI Interpretation Engine.

The long-term goal is to become one of the most trusted online references for Bazi, Five Elements, Ten Gods, Heavenly Stems, Earthly Branches, and related Chinese metaphysics systems.

## 3. Core Principles

### Principle 1: Knowledge First

Every feature must strengthen the knowledge system.

Tools bring traffic.  
Knowledge creates long-term value.

### Principle 2: AI Explains, AI Doesn't Invent

AI should explain existing metaphysical concepts, not fabricate unsupported claims.

AI-generated reports should reference Oneming Knowledge whenever possible.

### Principle 3: Everything Should Be Connected

No isolated pages.

Every concept should connect to:
- related concepts
- guides
- tools
- user reports

### Principle 4: Build Assets, Not Just Features

A feature is useful today.

An asset becomes more valuable over time.

Oneming prioritizes:
- knowledge objects
- internal linking
- structured data
- reusable explanations
- AI-readable content

## 4. Product Layers

Oneming has four product layers:

### Tools

Interactive calculators and AI reports.

Examples:
- Bazi Calculator
- Five Elements Calculator
- Lucky Number
- Name Analysis

### Knowledge

Structured encyclopedia pages.

Examples:
- Zhengguan
- Qisha
- Wutu
- Xiyongshen
- Strong Day Master

### Guides

Educational long-form tutorials.

Examples:
- How to Read a Bazi Chart
- How to Find Your Favorable Element
- What Does Day Master Mean?

### AI Engine

The AI layer that interprets user input using Oneming's own knowledge base.

## 5. Long-Term Goal

Oneming should become a trusted AI-native knowledge platform for Chinese metaphysics.

The goal is not to generate many pages quickly.

The goal is to build a system that becomes more accurate, more connected, and more trusted over time.
